# Security Policy

## Secrets

Do not commit API keys, OAuth tokens, bearer tokens, platform cookies, exported private customer data, or local research files.

Use `.env.local` for local credentials. The file is ignored by Git.

## Supported Use

This project is intended for:

- public data collection where allowed
- official APIs
- platform exports
- manually collected research data

It is not intended for bypassing login walls, anti-bot systems, paid access restrictions, or platform terms.

## Reporting Issues

If you find a security problem, open a private report if the repository host supports it, or contact the maintainer directly. Avoid posting live secrets or sensitive datasets in public issues.
